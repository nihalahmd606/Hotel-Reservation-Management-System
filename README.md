
# Hotel Reservation Management System

## Overview
MySQL database project for managing hotels, rooms, customers, reservations, and payments using SQL and relational database concepts.

## Features
- Hotel Management
- Room Management
- Customer Records
- Reservation Tracking
- Payment Management
- SQL Queries & Analysis

## Technologies Used
- MySQL
- SQL
